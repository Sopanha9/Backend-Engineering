A complete hands-on backend engineering course using **Node.js + Express.js**.

| EP03 | [Middleware & Validation](./episodes/ep03-middleware-validation) | Custom middleware, input validation |
| EP04 | [Authentication & Authorization](./episodes/ep04-auth) | JWT, sessions, RBAC |
| EP05 | [REST, CRUD & Controllers](./episodes/ep05-rest-crud-controllers) | REST best practices, controller pattern |
| EP06 | [Database](./episodes/ep06-database) | SQL, ORM, migrations |

## How to use this repo

Each episode folder contains:
- `README.md` — full explanation of concepts (follow along with the video)
- `src/` — working code you can run

```bash
cd episodes/ep01-http-and-routing
npm install
node src/index.js
```
