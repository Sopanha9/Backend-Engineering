How to use this repo
Each episode is self-contained — clone once, run any episode independently.
01
Clone the repo
Get all 18 episodes on your machine.
git clone https://github.com/Sopanha9/Backend-Engineering
cd Backend-Engineering
02
Pick an episode
Navigate to any episode and install its dependencies.
cd episodes/ep01-http-and-routing
npm install
03
Read the README
Full explanation with code examples. Follow along with the video.
# Each episode has a README.md
# with concepts + code examples
code README.md
04
Run and experiment
Working code you can run, break, and learn from.
npm start
# → Server running on :3000
curl localhost:3000/api/users
